# 👨‍👩‍👧 Mi PAU — Versión Personal (uso libre)

Versión **gratuita y sin cuentas** para uso personal de tu hija y sus amigos.

## Características
- ✅ Doble clic en `index.html` o cualquier hosting estático
- ✅ Datos en `localStorage` (cada dispositivo independiente)
- ✅ PWA instalable, modo offline
- ✅ IA opcional con claves propias del usuario (botón 🤖)
- ✅ Sync entre dispositivos por **QR/JSON** (sin servidor)
- ❌ Sin login, sin pagos, sin tracking

## Compartir con sus amigos
1. Sube esta carpeta a hosting estático gratis:
   - Netlify Drop: https://app.netlify.com/drop (arrastra y suelta)
   - GitHub Pages, Vercel, Cloudflare Pages
2. Comparte la URL por WhatsApp
3. Cada uno la añade a la pantalla de inicio (PWA)

## Diferencias con la versión "producto" (carpeta `../producto/`)

|  | Personal | Producto |
|---|---|---|
| Login / cuentas | ❌ | ✅ |
| Sync en nube | manual (QR) | automática |
| Pagos | ❌ | ✅ Stripe |
| Páginas legales / RGPD | ❌ | ✅ |
| Panel admin | ❌ | ✅ |
| Apps stores | PWA | PWA + Play + App Store |
| Coste mensual | 0 € | 25-250 € |
